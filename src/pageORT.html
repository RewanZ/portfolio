Mes deux plus improtants projets réalisés en CFA sont les projets HelpOrt réalisés en deux langages : php et java. Helport est une application d'aide aux étudiants, Dans cette application, les élèves de deux niveaux d'étude supérieur ou plus peuvent venir en aide à des étudiants demandant du soutien sur certains chapitres de différentes matières.

HelpOrt Web

Voici mon application Help Ort web, développée en PHP à l'aide du Framework Symfony, c'est un ensemble d'outils et de logiciels visant à faiciliter la création d'applications.

Sur cette page l'utilisateur, un administrateur, peut créer une matière, une sous matière ainsi qu'une salle.

Sur cette page l'utilisateur, un étudiant, peut créer une demande de soutien et mettre à jour ses compétences.

Help Ort Java

Voici mon application Help Ort applicatif, développée en Java, ici aucun framework n'a été utilisé, cepedant j'ai utilisé SceneBuilder pour créer l'inferface. Sur cette page l'utilisateur, un administrateur, peut créer une matière, une sous matière ainsi qu'une salle.
